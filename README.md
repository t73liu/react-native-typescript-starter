# React-Native TypeScript Starter

## Features

* UI Components
  * native-base
* Navigation
  * react-navigation
* Formatting
  * prettier
* Code Quality
  * tslint
  * tslint-react
* Testing
  * jest

## Setup

1.  Install NodeJS 8.X
2.  Install Expo on Phone
3.  Clone the repository
4.  npm/yarn install
5.  Refer to create-react-native-app [documentation](CRNA.md) for running app
